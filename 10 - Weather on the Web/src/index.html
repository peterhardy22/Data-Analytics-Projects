Latitude vs. X:

This project was conducted to see if we could find a relationship between a cities latitude and its weather patterns. Using OpenWeatherMap API to collect data from 543 cities, I was able to retrieve multiple variables to test this theory with.

Then Pandas was used to organize said weather data, and Matplotlib was used at the end to visualize the data. This webpage acts as a window into the research I have conducted. On the following pages you will find visualizations as well as all the data used to complete this project. Enjoy!
